Class Sistemi v1
- Explorer varsayılan class olacak.
- Terkedilmiş köy aşamasına geçen oyuncular için uzmanlaşma kitaplarının bulunması ve belirli eylemlerde uzmanlaşma yani class sistemi yapılacak.
- Explorer, Figther, Blacksmith, Tailor, Warrior, Miner classları olacak.
- Seçilen classa göre bazı özel recipeler açılacak. 
-- Örnek Tailor için özel kıyafetler, Miner için özel kazmalar, Fightler için özel kılıç, Blacksmith için özel ore işleme, Warrior için özel silah gibi..
- Her class kendine uygun şekilde bonuslar sağlayacak.
